Welcome

Welcome to the GameDevForge!
I'm Lewis a Senior Gameplay Developer currently working at BULKHEAD. I have put together this learning resource for anyone looking to become a game developer. We start with the very basics of understanding the core principles of programming and work up to more advanced features. See the modules page for more information. Good luck!

Java

Learn the very basics using Java. Create a basic 2D game engine using java starting from a simple hello-world project.

GitHub

Learn GitHub source control. How to store, manage and branch your projects. We will use GitBash and/or SourceTree to manage your Git repositories.

C++

Learn C++ development. How to manage memory, more complex computer architecture and much more! Create a 3D game engine using DirectX and Vulkan.

Visual Studio

Learn the Visual Studio IDE. General project management and config leading to more advanced features and debugging pratices.

Unreal Engine 5

Learn how to make games in Unreal Engine 5. How to structure a game project, create re-usable assets and plugins and learn the latest game development practices like data driven development.

Perforce

Learn how to use Perforce source control. How to host and manage your own Perforce server such as shelves, streams, tools etc.
